FAQ

Questions buyers ask.

Works without a Gemini API key?
Yes. The codebase ships with a regex + keyword fallback that catches the dominant U.S. opt-out patterns (STOP, UNSUBSCRIBE, "take me off", END, CANCEL, etc.). Gemini is only used if you set a real key. The dispatcher auto-falls-back if the Gemini API is unreachable.
Can it run on Postgres?
Yes — set DATABASE_URL=postgresql+asyncpg://USER:PASS@HOST:5432/DBNAME and the same SQLAlchemy ORM will pick it up. Migrations are handled at startup.
What if Twilio sends a malformed body?
It's acknowledged with a 200 + empty TwiML response so Twilio doesn't retry-bomb you. Strict-mode refuses unsigned requests entirely (no fallback).
Can I add another CRM after I buy?
Yes. Drop a method onto IntegrationDispatcher (e.g. _dispatch_to_activecampaign) and add an <option> to the new-integration form in the dashboard. Total ~30 minutes of dev.
